# Texturas — Youtube Disks Mod

Las texturas en esta carpeta son **placeholders** (16×16 PNG con damero de color sólido sobre negro) para que el modelo y el render del mod funcionen mientras la diseñadora produce las definitivas.

## Items (`textures/item/`)

| Archivo                | Qué es                                                                  |
| ---------------------- | ----------------------------------------------------------------------- |
| `blank_disk.png`       | Disco vacío listo para grabar. Inspiración: vinilo virgen.              |
| `recorded_disk.png`    | Disco con una canción grabada. Inspiración: música vanilla con un toque de YouTube (acento rojo / play button). |

## Bloques (`textures/block/`)

| Archivo                       | Qué es                                                                       |
| ----------------------------- | ---------------------------------------------------------------------------- |
| `disk_recorder_top.png`       | Cara superior del Disk Recorder (slot donde entra el disco).                 |
| `disk_recorder_side.png`      | Caras laterales del Disk Recorder.                                           |
| `disk_recorder_bottom.png`    | Cara inferior del Disk Recorder.                                             |

## Specs para la diseñadora

- **Formato:** PNG, 16×16 píxeles, paleta libre.
- **Estética:** Pixel art al estilo Minecraft vanilla.
- **Mantener los nombres de archivo exactos** — el mod los referencia por path.
- Sumar variantes (animadas con `.mcmeta`, emisivas, etc.) es opcional y se charla aparte.

Reemplazar los placeholders cuando estén listos los assets finales.
